# neovim config

Personal Neovim setup built on [LazyVim](https://lazyvim.github.io).

## Features

- **Theme** — Tokyo Night (night variant)
- **Dashboard** — custom ASCII art header via snacks.nvim
- **Typing practice** — [Typr](https://github.com/nvzone/typr) (`:Typr`, `:TyprStats`)

## Language Support

Dart, Python, Rust, TypeScript, Tailwind, Markdown, LaTeX, JSON, YAML, TOML

## Install

```bash
git clone <repo> ~/.config/nvim
nvim  # lazy.nvim bootstraps everything on first launch
```
